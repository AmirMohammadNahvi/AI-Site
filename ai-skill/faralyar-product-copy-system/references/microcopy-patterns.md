# Microcopy patterns

## CTA examples
Good:
- شروع گفتگو
- امتحان رایگان
- انتخاب پلن
- ادامه پرداخت
- ذخیره تغییرات

Avoid:
- کلیک کنید
- ثبت
- ادامه
- ارسال

## Error pattern
Structure:
- what failed
- why if known
- what to do next

## Empty state pattern
Structure:
- what this area is
- why it matters
- what the user should do now
