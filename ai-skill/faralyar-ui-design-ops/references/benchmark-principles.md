# Benchmark principles for FaralYar

Use these products as pattern sources, never as direct layout templates.

## ChatGPT
Useful principles:
- low-friction start
- one dominant primary action
- reduced visual clutter
- calm composition
- clear input-first behavior

## Perplexity
Useful principles:
- trust-oriented answer experience
- continuity across threads and history
- cleaner reading rhythm
- stronger follow-up feeling

## AvalAI
Useful principles:
- explicit product explanation
- strong visibility for models and pricing
- Persian-first product communication
- visible capability listing

## Houshan
Useful principles:
- capability discoverability
- multiple clear entry points into tools
- guided exploration of use cases

## Synthesis for FaralYar
FaralYar should combine:
- ChatGPT clarity
- Perplexity trust and continuity
- AvalAI product communication
- Houshan discoverability

Do not make the result look derivative.
